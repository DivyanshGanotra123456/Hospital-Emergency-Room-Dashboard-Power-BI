# Hospital Emergency Room Dashboard using Power BI

## Overview

This project is an interactive Power BI dashboard built to analyze the performance of a hospital emergency room. The dashboard transforms raw patient data into meaningful insights that help monitor hospital operations, patient flow, waiting times, referrals, and patient satisfaction.

The report is designed to provide hospital administrators and decision-makers with a clear view of emergency room performance through interactive visualizations and dynamic filtering.

## Problem Statement

Managing emergency room operations involves tracking a large amount of patient data every day. Without proper reporting, it becomes difficult to identify delays, monitor patient satisfaction, analyze referrals, and understand patient demographics.

This dashboard was created to solve these challenges by presenting important hospital metrics in a simple and interactive format.

## Dashboard Pages

### Monthly View

The Monthly View provides a detailed analysis for a selected month. It includes:

- Total Number of Patients
- Average Waiting Time
- Patient Satisfaction Score
- Number of Patients Referred
- Admission Status
- Patients Seen Within 30 Minutes
- Referrals by Age Group
- Patient Distribution by Gender
- Department Referral Analysis
- Patient Race Distribution
- Patient Arrival Analysis by Day and Hour

### Consolidated View

The Consolidated View provides an overall summary of hospital performance across all available months. It helps compare trends and monitor overall emergency room performance.

### Patient Details

The Patient Details page contains patient-level information including:

- Patient Name
- Age
- Gender
- Admission Status
- Department Referral
- Waiting Time
- Admission Date
